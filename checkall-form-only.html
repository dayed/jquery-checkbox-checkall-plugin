关于 Checkall Form Only JS

Checkall Form Only JS 可以帮助我们快熟的选择一个Form表单中的多选按钮，要使用这个功能，需要引用jQuery.js 和 checkall.js文件

下载 checkall-form-only.js 下载 jQuery.js

要使用更多功能的checkbox 请参考 Checkall JS

示例

#1: 使用checkbox

下面这个例子中，我们用了一个 checkbox 作为全选的按钮，只需要给这个 checkbox 加 data-toggle="checkall" 属性或者 class="checkall-toggle" 即可

使用方法

Checkall JS是基于jQuery的插件，使用前，需要先引入 jQuery1.9+ 。

创建一个form表单， 给表单中用于全选的input 添加一个data-toggle="checkall"属性。

或者给表单中用于全选的input 添加一个class="checkall-toggle" 。
